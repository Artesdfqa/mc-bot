# MC Bot — Phoenix PE (Render)

Minecraft Bedrock бот с веб-интерфейсом. Полностью на Render — твой компьютер не нужен.

## Структура

```
mc-bot-render/
├── public/
│   └── index.html   # Веб-интерфейс
├── server.js        # Express сервер + управление ботами
├── package.json
└── README.md
```

## Деплой на Render (бесплатно)

1. Залей папку на **GitHub** (создай репозиторий, загрузи файлы)

2. Зайди на [render.com](https://render.com) → **New** → **Web Service**

3. Подключи свой GitHub репозиторий

4. Настройки:
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Plan**: Free

5. Нажми **Deploy** — через 2-3 минуты сайт будет доступен

## Использование

- Открой выданный Render URL
- Введи ник и пароль → нажми **Запустить**
- Бот подключится к серверу и пройдёт GUI авторизацию
- В разделе **Активные боты** видны все запущенные боты
- Кнопка **ЛОГ** — смотреть подробный лог бота
- Кнопка **СТОП** — остановить бота

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | /api/start | Запустить бота `{ username, password }` |
| POST | /api/stop  | Остановить бота `{ username }` |
| GET  | /api/bots  | Список всех ботов |
| GET  | /api/logs/:username | Логи конкретного бота |

## Важно

- На бесплатном Render сервис "засыпает" после 15 минут неактивности
- Боты живут пока сервис запущен
- Для постоянной работы можно взять платный план ($7/мес)
